# Minimalistic_Drawing_Panel

A clean and lightweight drawing add-on for **Anki** that lets you write, sketch, highlight, and solve directly on your cards while studying.

Perfect for:
- quick handwritten notes
- solving math and science problems
- underlining important ideas
- drawing diagrams
- studying with a mouse, touch screen, or stylus

The add-on is designed to stay **minimal, fast, and distraction-free**.

---

## Features

- Write and draw on any card during review
- Works with **mouse, touch, pen, and stylus**
- **Surface Pen support**
  - pen tip = write
  - eraser tip = erase automatically
- **Line eraser and dot eraser**
- **5 thickness levels**
- Hold thickness dots to open **colour picker**
- Configurable **side button action**
- Auto-clear drawing on next card
- Show / hide toolbar
- Lock drawing area to prevent accidental resize
- Automatic dark / light mode support

## Installation

### Option 1 — Manual install from GitHub
1. Click the green **Code** button
2. Select **Download ZIP**
3. Extract the ZIP file
4. Copy the add-on folder into your Anki add-ons folder:

### Windows
```text
C:\Users\YOUR_NAME\AppData\Roaming\Anki2\addons21\
```

### macOS
```text
~/Library/Application Support/Anki2/addons21/
```

### Linux
```text
~/.local/share/Anki2/addons21/
```

5. Restart Anki

Or https://ankiweb.net/shared/info/1062267970?cb=1775912735881

## How to use

- Click the **pen icon** to activate drawing
- Click again to collapse and deactivate
- Use thickness dots to change line size
- Hold thickness dots to choose colour
- Hold eraser icon to switch erase mode
- Use the tiny dot to show / hide toolbar
- Open **Tools → Minimalistic Drawing Panel Settings** for more options
